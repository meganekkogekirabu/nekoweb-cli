nekoweb-cli is a CLI for Nekoweb inspired by [Ratakor/nk](https://github.com/Ratakor/nk).

# Installation

## Arch User Repository
Available as `nekoweb-cli-bin`.

## Build from source
```console
$ git clone https://github.com/meganekkogekirabu/nekoweb-cli
$ cd nekoweb-cli
$ cargo build --release
```
